### **目录说明：**

```
src：
	api ：   NodeJs接口
	assets：   MongoDB的数据库
	component：   组件
	components：   主要的5大页面
	img：   图片
	js：    js文件
	pages：  其他跳转的页面
	router：  Vue的路由
	style：   css文件
	
	main.js 是Vue的公共库
	server.js 是Node.js的 静态服务器文件
```
